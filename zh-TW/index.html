下載 Servo 開發者預覽版本

Servo 是一款專為應用軟體和嵌入式應用而設計的現代化的高效能、平行化瀏覽器引擎


這些預先版本，讓開發者可以不用在 本地端編譯 的情況下試用 Servo 以及 回報 issues

目前請不要使用 Servo 來登入銀行帳號！ 目前我們釋出第一版的開發者預覽版本，我們將進行正式的安全性稽核， 並且透過現存的函式庫還有 Rust 語言來增進安全性 -- 將會有更多資訊釋出！

macOS 安裝設定

  • 點擊上方「 macOS Build 」來下載最新預覽版
  • 打開下載的 `servo-latest.dmg` 檔案
  • 拖曳 Servo.app 到應用程式資料夾
  • 雙擊 Servo.app 來執行 Servo

注意： Servo.app 目前沒有簽章，所以 Mac 的用戶會看到應用程式跳出「無法打開『Servo.app』，因為它來自未識別的開發者。」。 並出現「您的安全性偏好設定僅允許安裝來自 App Store 和已識別開發者的 App。」。 請到「系統偏好設定」的「安全性與隱私」來解鎖，並執行程式。

Linux 安裝設定

  • 點擊 「 Linux Build 」 按鈕
  • 下載 .TAR.GZ 壓縮檔案
  • 解壓縮(`tar zxf servo-latest.tar.gz`)
  • `cd servo`
  • `./servo` 來執行 Servo
